# Completed Execution Plans

Archive finished or intentionally closed plans here.

## Rules

- Preserve the original filename when moving a plan from `active/`.
- Add a short completion note or outcome section if the plan changed during execution.
- Link completed plans from PRs, design docs, product specs, or tech-debt items when they remain useful references.

Completed plans are historical records. Start a new plan for new work rather than reopening an old one in place.
