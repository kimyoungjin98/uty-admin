# Active Execution Plans

Use this folder for plans tied to work that is in progress or ready to start.

## Naming

- Prefer `YYYY-MM-DD-short-title.md`
- Keep titles specific enough to scan in a directory listing

## Usage

- Create the plan here before starting non-trivial work.
- Link the plan from the PR, design doc, product spec, or tech-debt item that drove the change.
- Move the file to `../completed/` once the work is done or intentionally stopped.

See [docs/PLANS.md](../../PLANS.md) for the required sections.
