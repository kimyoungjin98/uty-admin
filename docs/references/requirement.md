1. 간편 원클릭 발주/결제금액 자동계산 시스템 도입
                           
1) 계정선택 - 업체등록
1-1) 업체(상품)링크 입력 - 자동정보 불러오기 - 작성유형 선택 (후기성/정보성) - 키워드 입력
                                                
1-2) 배포상품 선택 - 클릭시 주문 상품별 나열 및 상품별 금액표기 (부가세미포함 기준)

1-3) 옵션 추가 여부 선택 ex) 24년 이전블로그배포 / 일반배포

1-4) 수량 및 진행 일자 선택 - 금액 자동계산 (옵션추가까지 포함)

1-5) 배포 사용 이미지 압축파일 등록
 (이메일 첨부가 아닌 신청서에서 바로 등록하면 관리자가 확인 가능)

*결제방식* 포인트 충전식 or 접수 건당 별도 입금계좌 표기 및 입금 확인 후 관리자 승인 방식
